# ResQNet – AI Disaster Detection & Emergency Response Platform

An AI-powered disaster management platform designed for real-time disaster monitoring, emergency response coordination, and resource management. The system helps detect disasters, optimize rescue operations, and provide live analytics through an interactive dashboard.

---

## Features

* AI-based disaster detection and monitoring
* Emergency alert and response system
* Interactive real-time dashboard
* Rescue route optimization using maps
* Resource tracking and management
* Disaster analytics and visualization

---

## Tech Stack

* React.js
* TypeScript
* Tailwind CSS
* MongoDB
* OpenCV
* YOLOv8
* Google Maps API

---

## Functionalities

* Real-time disaster monitoring
* Fire and flood detection
* Emergency notification system
* Safe-zone navigation
* Rescue team coordination
* AI-driven severity analysis

---

## Deployment

* Frontend: Vercel
* Backend Services: Render

## Future Improvements

## Project Objective

The goal of this platform is to improve disaster response efficiency through AI-powered monitoring, analytics, and emergency coordination systems.
